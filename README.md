# Description

This project is a simple web authenticator app created with React and Firebase. It can be implemented in a variety of web platforms to safely authenticate users as they try to sign into their accounts. 

## Tech Stack

- React
- React Router Dom: For declarative routing.
- Protected Routes: A protected route component to secure important routes like the account page.
- Tailwind: Visual styling where necessary.
- Firebase: Authentication platform and input validation feature.
- React Icons: Password visibility feature when the eye icon is clicked.
- Context
